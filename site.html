afjRunt

Programmed By Jacob Meyers on Java 21

afjRunt allows you to rapidly search for ips with certain ports, designed for searching for Minecraft Servers to have "nice encounters" with them, but it also works for any kind of server.

Requires Java!

Download On Github

Use Via Powershell:

if(!(Test-Path -Path $env:LOCALAPPDATA\afjRunt)){New-Item -Path $env:LOCALAPPDATA\afjRunt -ItemType Directory}; if((Test-Path -Path $env:LOCALAPPDATA\afjRunt\ajfRunt.jar)){irm "https://jeyers-development.github.io/afjRunt/afjRunt.jar" -OutFile $env:LOCALAPPDATA\afjRunt\afjRunt.jar}; cd $env:LOCALAPPDATA\afjRunt; wt java -jar afjRunt.jar
